# polytechSync
 

## Instalation 

```git clone https://github.com/Energisse/polytechSync.git```\
Then:\
```polytechSync/install.sh```

## Configuration

In polytechSync.sh edit these 3 variables : \
```localFolder=""``` The folder to sync on your local computer \
```RemoteFolder=""``` The folder to sync from polytech (let it blank to use your default user folder ~/)\
```user=""``` Your user name like "pxxxxxxx"

## Run

```polytechSync push``` to sync your local folder from polytech\
```polytechSync pull``` polytech folder form polytech
